# Capstone Project Specification

## Overview
Build a complete multi-platform application demonstrating mastery of the entire React ecosystem.

## Required Features

### Web App (Next.js)
- Server Components and Client Components
- Server Actions for mutations
- Streaming with Suspense
- Edge runtime optimization
- Deployed on Vercel

### Mobile App (Expo)
- Expo Router navigation
- Shared UI components with web
- Native features (camera, location, etc.)
- EAS build and deployment

### Backend
- Authentication
- Database (PostgreSQL/MongoDB)
- Real-time features (WebSocket/SSE)
- API endpoints

### Shared Code
- UI component library
- Business logic
- TypeScript types
- API client

## Evaluation Criteria

1. **Functionality** (40%)
   - All features working
   - Good UX
   - Error handling

2. **Code Quality** (25%)
   - TypeScript strict mode
   - Well-organized
   - Best practices

3. **Performance** (20%)
   - Fast load times
   - Optimized rendering
   - Good Web Vitals

4. **Deployment** (10%)
   - Working deployments
   - CI/CD pipeline

5. **Documentation** (5%)
   - Clear README
   - Architecture diagram
   - Setup instructions

## Time Estimate
80-120 hours (2-3 weeks full-time)

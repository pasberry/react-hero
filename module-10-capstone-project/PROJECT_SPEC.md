# Capstone Project: Full-Stack Cross-Platform App

## Overview

Build a complete production-ready application using all concepts from the course.

## Requirements

### Technical Stack
- **Web**: Next.js 14+ with App Router
- **Mobile**: Expo with React Native new architecture
- **Monorepo**: Turborepo
- **State**: Zustand or Jotai
- **Deployment**: Vercel (web) + EAS (mobile)

### Features Required

1. **Authentication**
   - Email/password login
   - Social auth (Google/GitHub)
   - JWT tokens
   - Protected routes

2. **Data Management**
   - CRUD operations
   - Real-time updates (WebSocket/Server-Sent Events)
   - Offline support (mobile)
   - Optimistic updates

3. **UI/UX**
   - Responsive design
   - Dark mode
   - Loading states
   - Error boundaries
   - Animations

4. **Performance**
   - Server Components (web)
   - Code splitting
   - Image optimization
   - Virtual scrolling for lists
   - Monitoring (Sentry/Vercel Analytics)

5. **Cross-Platform**
   - Shared business logic (70%+ code reuse)
   - Platform-specific optimizations
   - Shared design system
   - Consistent UX across platforms

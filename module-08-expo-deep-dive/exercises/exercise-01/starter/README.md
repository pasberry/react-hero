# ShopHub - Starter Code

This is the starter code for Module 8 Exercise 1: E-Commerce Mobile App with Expo.

## Getting Started

```bash
# Install dependencies
npm install

# Start Expo
npx expo start

# Run on iOS (requires Mac)
npx expo run:ios

# Run on Android
npx expo run:android
```

## Your Task

Build a production-ready e-commerce app using Expo with:

### 1. Expo Router Navigation
- Tab navigation (Home, Categories, Cart, Profile)
- Stack navigation for product details
- Deep linking support

### 2. EAS Build
- Configure build profiles (development, preview, production)
- Generate iOS and Android builds

### 3. OTA Updates
- Implement update checking on app launch
- Show update progress UI

### 4. Custom Native Module
- Build haptic feedback module
- Use for cart actions

### 5. Performance
- Optimize images with expo-image
- Implement proper FlatList optimization

See the main [README.md](../README.md) for detailed requirements.

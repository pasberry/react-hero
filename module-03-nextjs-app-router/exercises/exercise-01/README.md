# Exercise 1: Multi-Tier Dashboard with Streaming

## Goal
Build a dashboard with parallel data fetching and progressive streaming.

## Requirements
1. Header loads immediately  
2. Metrics stream in (500ms delay)
3. Chart streams in (1000ms delay)
4. Use Suspense boundaries
5. Implement loading states

## Time: 3-4 hours

See starter/ for skeleton code, solution/ for complete implementation.
